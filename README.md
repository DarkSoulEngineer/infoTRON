Open labs for the **infoTRON 2026 mechatronics program**.  
The focus is on approachable Arduino/ESP32 exercises that give students **quick wins**, then encourage iteration and exploration.

---

## Features
- **LED Demos:** Quick GPIO experiments and simple animation patterns.  
- **Web Demos:** ESP32 Async Web Server with static HTML, CSS, and JS assets. Includes screenshots for guidance.  

---

## Requirements
- ESP32 development board  
- Breadboard, jumper wires, LEDs, resistors  
- Arduino IDE with **ESP32 board package**  
- Optional: [ESPAsyncWebServer](https://github.com/me-no-dev/ESPAsyncWebServer) and LittleFS plugin for web demos  

---

## Getting Started
1. Pick a lab folder (`LED/` or `Web/`)  
2. Open the corresponding `.ino` sketch in Arduino IDE  
3. Connect your ESP32 board and upload  
4. For web demos: access the ESP32 IP in a browser to interact with the UI  

> Feel free to adapt pinouts, timings, or UI text to match your classroom kit.
